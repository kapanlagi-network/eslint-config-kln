Opinionated ESLint configuration
=======================================


## Installation

Using [npm](https://www.npmjs.com):

```
npm install eslint-config-kln
```


## Documentation

Extends your [ESLint configuration](http://eslint.org/docs/user-guide/configuring.html):

```js
{
  extends: ['eslint-config-kln']
}
```
